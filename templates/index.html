Rainfall Susceptibility and Return Period

This model estimates extreme rainfall events for Kheda District using statistical methods like the Gumbel Distribution and Fisher-Tippett Distribution. The model predicts return periods for rainfall events ranging from 100mm to 300mm in a single day.

These extreme event predictions help in understanding the likelihood of rare and intense rainfall, which is crucial for disaster management and agricultural planning.
